# Directorio de Plantillas

Este directorio contiene plantillas base y ejemplos de datos.

## Tipos de Plantillas

- **Plantillas Excel Base**: Ficheros `.xlsx` con formato predefinido
- **Plantillas de Datos**: Archivos YAML/JSON con estructura de datos ejemplo
- **Estilos**: Definiciones de estilos y formatos reutilizables

## Uso

Las plantillas se utilizan como base para generar nuevos ficheros Excel parametrizados.
Puedes crear nuevas plantillas o modificar las existentes según necesidades.

## Ejemplos

- `viviendas_template.xlsx` - Plantilla para listado de viviendas
- `data_example.yaml` - Ejemplo de estructura de datos
- `styles_config.yaml` - Configuración de estilos y formatos
