# Directorio de Configuración

Este directorio contiene los archivos de configuración para el generador de Excel.

## Archivos

- `settings.yaml` - Configuración principal del sistema
- `parameters.yaml` - Parámetros parametrizables para plantillas
- `company.yaml` - Información de la empresa constructora

## Notas

- Los archivos de configuración con credenciales sensibles (`secrets.yaml`, `credentials.yaml`) están excluidos en `.gitignore`
- Editar estos archivos mediante el script `config_manager.py` o manualmente
